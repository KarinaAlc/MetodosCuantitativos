  1. Sintaxis en Python

En Python, la sintaxis es el conjunto de reglas que determinan cmo debe escribirse el cdigo, es decir, la manera en la que hablaremos con Python.

6.1. Palabras reservadas Python tiene palabras reservadas que son reconocidas por el programa y estn diseadas para gatillar funciones especficas y que por ende no deberan ser usadas para la creacin de variables, estas son:

and as assert break class continue def del elif else except False finally for from global if import in is lambda None nonlocal not or pass raise return True try while with yield

6.2. Variables

Una variable es un nombre que hace referencia a un valor almacenado en memoria.Se crean con una sentencia de asignacin: nombre = valor. El tipo de la variable depende del valor asignado (int, str, etc.). Es necesario inicializarlas antes de usarlas, o se producir un NameError.

6.2.1. Tipos de datos Se refiere a la naturaleza de los datos que usamos para comunicarnos con Python y que pueden ser reconocidos por el programa:

  • int (Entero): Nmeros sin parte decimal. Ej.: 17
  • str (Cadena): Texto entre comillas simples o dobles. Ej.: "Qu tal!"
  • float (Punto flotante): Nmeros con parte decimal. Ej.: 3.14 o pi
  • bool (Booleano): Valores lgicos True o False.
  • list (Lista): Secuencia mutable de elementos. Ej.: [1, 2, 3]
  • dict (Diccionario): Pares clave-valor. 
    Ej.: {"nombre": "Ana", "edad": 25}
  • tuple (Tupla): Secuencia inmutable. Ej.: (1, 2, 3)
  • NoneType: Representa ausencia de valor. Ej.: None

6.3. Sensibilidad a maysculas y minsculas: 'Variable' y 'variable' son nombres distintos.

6.4. Indentacin obligatoria: Se usan espacios (recomendado 4) o tabulaciones para definir bloques.

6.5. Comentarios:

  • Una lnea: usando "#".
  • Varias lneas: usando comillas triples

6.6. Nombres de variables:

  • Deben empezar con letra o guion bajo.
  • Pueden contener letras, nmeros y guion bajo, pero no espacios.
  • Evitar palabras reservadas (if, for, while, etc.).

6.7. Sentencias:

  • Generalmente una por lnea, pero pueden separarse con ';' o unirse con '\'.

6.8. Cadenas de texto:

  • Se definen con comillas simples (' '), dobles (" ") o triples para varias lneas.

6.9. Bloques de cdigo:

  • No se usan llaves '{}' como en otros lenguajes, sino indentacin.

6.10. Uso de parntesis:

  • Para llamadas a funciones, agrupar operaciones o definir tuplas.
